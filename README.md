# atb
![Лого](images/AtB.png)

ATB is free-use library for developing telegram bots  
The abbreviation ATB means asyncio telegram bot  
Version: 0.1

# Changelog

- Added bot  
- Added decorator  
- Added message  
- Added callback query  
- Added inlinebutton & inlinemarkup (miniapp include)  
- Added replybutton & replymarkup

# Installing (locally)

```sh
cd atb & pip install .
```
